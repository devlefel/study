# Math and Logic Puzzles

## Prime Numbers

- **Divisibility**: A number $x$ divides $y$ if $y \% x == 0$.
- **Prime**: A number > 1 divisible only by 1 and itself.
- **Checking Primality**: Iterate from 2 up to $\sqrt{N}$.
- **Sieve of Eratosthenes**: Efficient way to generate primes up to $N$.

## Probability

- **P(A and B)**: $P(A) * P(B)$ (if independent).
- **P(A or B)**: $P(A) + P(B) - P(A \text{ and } B)$.

## Logic Puzzles

- **Start Small**: Try $N=1, 2, 3, 4, 5$. Look for patterns.
- **Worst Case Minimization**: "Minimize the maximum time".
